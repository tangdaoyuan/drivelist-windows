# drivelist-windows

> Get driver list in windows


## Install

```bash
npm install @suger-tdy/drivelist-windows -g
```

## Usage

```bash
@suger-tdy/drivelist-windows
```

## Programmatic API

```ts
import { list } from '@suger-tdy/drivelist-windows'

const drivers = await list()

console.log(drivers)
```
